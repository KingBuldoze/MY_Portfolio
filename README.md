# Personal Portfolio

A premium, cinematic portfolio experience inspired by high-tech HUD systems and neural link interfaces.

## Features
- **Neural Link Interface**: High-performance cinematic loading sequence.
- **HUD Dashboard**: Real-time system telemetry and interactive HUD elements.
- **Scroll-Linked Animations**: Frame-by-frame video playback synchronized with scroll progress.
- **Premium Aesthetics**: Dark mode, glassmorphism, and vibrant HSL-tailored accents.

## Tech Stack
- Next.js 14
- TypeScript
- Framer Motion
- Canvas API
- Vanilla CSS

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the project.
